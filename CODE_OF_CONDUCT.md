# Code Of Conduct

## Our Commitment

We want this project to be respectful, constructive, and welcoming to everyone.

## Expected Behavior

- Be respectful in discussions and reviews
- Give clear, technical, and constructive feedback
- Assume good intent and collaborate professionally
- Focus on improving the project, documentation, and verification quality

## Unacceptable Behavior

- Harassment, insults, or personal attacks
- Discriminatory or abusive language
- Repeated disruptive behavior in discussions or contributions

## Enforcement

Project maintainers may remove or reject contributions that do not follow this code of conduct.

## Scope

This code of conduct applies to issues, pull requests, code reviews, discussions, and other project interactions.
